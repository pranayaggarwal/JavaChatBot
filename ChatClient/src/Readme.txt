To run Chat Client in terminal browse to sc folder and type - java ChatClient 
or run project through IDE

>> Enter ip address of server, port number, your name in input dialogs
>> Type your messages and enter send button
>> By deafault sent messages will be broadcasted
>> To send private messages type - @<receievername> <message>
>> To send private messages to many users type - @<receievername1> <receivername2> ....   <receivernameN>#<message>
>> To see list of online users - Press list of users button
>> To quit chat room - type QUIT
